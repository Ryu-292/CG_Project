# WebGPU Interactive Ray Marching Scene Editor

3D scene editor with real-time shader editing, built with WebGPU and WGSL.

![Scene Editor Preview](./cg_project.gif)

## Live Demo

**[View Live Demo](https://Ryu-292.github.io/CG_Project/)**

## Overview

An interactive WebGPU-powered scene editor enabling real-time shader compilation, dynamic scene object manipulation, and professional-grade ray marching visualization. 

## Features

- Real-time shader editing with WGSL syntax highlighting
- Interactive scene panel for object property manipulation
- Color picker for material customization
- Ray marching rendering with smooth animations
- Change sphere radius and position dynamically
- Adjust torus radius, thickness, and position in real-time

## Tech Stack

- WebGPU - Modern GPU computing API
- WGSL - WebGPU Shading Language
- JavaScript - Application logic
- HTML & CSS - User interface
- Ray Marching - Advanced rendering technique

## Local Development

### Prerequisites

- Modern browser with WebGPU support (Chrome, Edge, Opera)
- Python 3.x
- Git

### Quick Start

```bash
# Clone the repository
git clone https://github.com/Ryu-292/CG_Project.git
cd CG_Project

# Start local development server
python -m http.server

# Open browser and navigate to
# http://localhost:8000
```
